## 👋 About Me

> 새로운 아이디어를 플레이 가능한 재미로 만들고,  
> 기능 구현에서 멈추지 않고 플레이 경험까지 개선하는 개발자를 목표로 합니다.

- Unity와 C#을 중심으로 게임 클라이언트와 플레이 시스템을 개발합니다.
- FSM, Event Channel, JSON 저장 시스템, UI 구조, 게임플레이 시스템 구현 경험이 있습니다.
- 직접 플레이하며 불편한 흐름과 지루한 지점을 찾고, 플레이 경험을 개선하는 과정을 중요하게 생각합니다.
- 코드 구조를 다시 살펴보고, 의존성을 줄이며, 수정과 확장이 쉬운 구조로 리팩토링하는 것을 지향합니다.
- 팀 프로젝트에서는 개인의 구현물보다 게임의 방향성과 완성도를 우선합니다.
